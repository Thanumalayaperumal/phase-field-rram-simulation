# Phase-Field Simulation of Conductive Filament Formation in RRAM Devices

## Overview

This project implements a finite-difference phase-field model in Julia to simulate conductive filament formation in Resistive Random Access Memory (RRAM) devices.

The model investigates electroforming dynamics through the coupling of phase-field evolution and electrostatic interactions.

---

## Objectives

- Model conductive filament nucleation and growth
- Study electroforming behavior in RRAM devices
- Analyze free-energy evolution during filament formation
- Reproduce key results reported in the literature

---

## Physics and Mathematical Model

The simulation is based on:

- Cahn–Hilliard Equation
- Double-Well Free Energy Functional
- Electrostatic Potential Equation
- Conductivity Interpolation Model

---

## Numerical Methods

- Finite Difference Method (FDM)
- Explicit Time Integration
- Iterative Laplace Solver
- 2D Phase-Field Simulation

---

## Features

- Conductive filament nucleation
- Filament growth and evolution
- Voltage distribution analysis
- Free-energy convergence monitoring
- Visualization of conductive pathways

---

## Tools and Technologies

- Julia
- Scientific Computing
- Phase-Field Modeling
- Numerical Simulation
- Data Visualization

---

## Results

The simulation successfully demonstrates conductive filament formation and reproduces qualitative trends reported in phase-field studies of RRAM devices.
